# Controllers API Reference (for frontend)

This folder contains FastAPI routers used by the backend. The files below document each controller's public routes, expected request shapes, authentication notes, and quick examples a frontend developer can use to call the API.

Files:

- `user.md` — Endpoints for user registration, login, profile, and token refresh (prefix: `/api/v1`).
- `admin.md` — Admin-specific endpoints (prefix: `/api/v1`).
- `root.md` — Root health/simple endpoint (`/`).
- `ws.md` — WebSocket endpoints used for real-time chat (`/ws/test` and `/ws/{user_id}`).

Open the individual files for details and examples.
